1. Comment highlighted code 	Ctrl + Shift + C
2. Uncomment highlighted code 	Ctrl + Shift + X 

3.  New file or project 	Ctrl + N 

4. Build 			Ctrl + F9
5. Run 				Ctrl + F10
6. Build and Run 		F9

*** Before using f9 to build and run, we have to save our file first. Otherwise, the code will throws an error.
